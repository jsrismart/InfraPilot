# Manual Mermaid Import to Lucidchart - Step-by-Step Guide

## You have an empty Lucidchart document

Your Infrastructure Diagram document was created successfully, but the diagram code needs to be imported manually. Follow these steps:

### Step 1: Get the Diagram Code
1. Go back to the InfraPilot application
2. Click **"üöÄ Export to Lucidchart"** button
3. A modal dialog will appear with the Mermaid diagram code
4. **Copy** the diagram code from the code box in the modal

### Step 2: Open Your Lucidchart Document
1. You can click the **"üìù Open in Lucidchart (Edit)"** button in the modal, OR
2. Visit your document directly at: `https://lucid.app/lucidchart/{DOCUMENT_ID}/edit`

### Step 3: Import the Mermaid Code
1. In your Lucidchart document, click the **File** menu (top-left corner)
2. Select **"Import"** from the menu
3. A dialog will appear asking what format to import
4. Select **"Mermaid Diagram"** or paste the code if prompted

### Step 4: Paste the Code
1. Paste the Mermaid diagram code you copied in Step 1
2. Click **"Import"** button
3. **The architecture diagram will automatically appear on your canvas!**

### Step 5: Customize (Optional)
1. The diagram will be imported with basic styling
2. You can now edit, rearrange, and customize the diagram in Lucidchart
3. Add colors, annotations, and formatting as needed

---

## Alternative: View the Diagram First

If you want to preview the diagram before importing:
1. Click **"üîç View Full Preview"** in the modal
2. This opens the diagram on **mermaid.live** where you can see how it looks
3. Then follow the import steps above

---

## Troubleshooting

**"Mermaid" option not visible in Import menu?**
- Make sure you're using a recent version of Lucidchart
- The Mermaid import feature is available in the newer versions

**Diagram didn't import?**
- Check that you copied the entire code block
- Make sure there are no extra spaces or characters
- Try the import again

**Need help?**
- Visit: https://mermaid.live to preview the code
- Copy fresh from the InfraPilot modal and try again
